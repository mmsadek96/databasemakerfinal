How to Setup Your Binance API Connection
Follow these steps to securely connect your trading bot to your Binance account.
1. Create API Keys on Binance
  1. Log in to your Binance account
  2. Navigate to "API Management" in your account settings
  3. Click "Create API" and complete the security verification
  4. Set an appropriate label for your API key (e.g., "Trading Bot")
  5. Enable reading information and spot & margin trading permissions
  6. Do NOT enable withdrawal permission unless absolutely necessary
2. Set Up IP Restrictions (Highly Recommended)
  1. Determine your server's IP address or your static IP
  2. Enter this IP in the "Restrict access to trusted IPs only" section
  3. For multiple IPs, separate them with commas in the Allowed IP Addresses field
  4. Note: If your IP is dynamic, you'll need to update the restrictions when it changes
3. Save and Test Your Configuration
  1. Enter your API key and secret in the fields above
  2. Click "Save Credentials" to store your API information
  3. Test the connection to ensure everything is working properly
  4. Begin monitoring your account balances and orders
Security Notice: Never share your API keys with anyone. If you suspect your keys have been compromised, immediately delete them on Binance and create new ones.
